# 2D Geometry Operations (Python)

A **menu-driven 2D Geometry Calculator** built using **Python and Object-Oriented Programming (OOP)**.

This project allows users to perform common geometric operations such as distance calculation, midpoint calculation, slope of a line, checking if a point lies on a line, and determining whether two lines intersect or are parallel.

This project was created to practice **Python programming, OOP concepts, and mathematical computations**.

---

# Features

The program supports the following operations:

1. Distance between two points
2. Distance of a point from the origin
3. Midpoint of two points
4. Slope of a line
5. Check if a point lies on a line
6. Check if two lines are parallel or intersecting
7. Exit the program

The program runs through a **menu-driven interface** where the user selects an operation and enters the required values.

# Example Calculation

Example: Distance between two points

```
Point 1: (2,3)
Point 2: (6,7)

Distance = √((x2-x1)² + (y2-y1)²)

```
