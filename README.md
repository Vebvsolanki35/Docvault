# 🗂️ DocVault

> Your documents. Private. Always.

A premium, **local-first Personal Document Manager** built as a Progressive Web App (PWA). All your documents stay on YOUR device — no server, no cloud, no tracking.

🔗 **Live Demo**: <a href="https://YOUR-VERCEL-LINK-HERE">https://YOUR-VERCEL-LINK-HERE</a>

---

## ✨ Features

- 🔐 **Password Protected** — Master lock with session persistence
- 👨‍👩‍👧 **People-Based Organization** — Manage docs for yourself and family
- 📁 **Smart Folders** — Identity, Education, Medical, Financial, Photos, Others
- 🔍 **OCR Text Extraction** — Extract text from images using Tesseract.js
- 📄 **PDF Viewer** — Render PDFs natively in-browser
- 📦 **ZIP Export** — Backup your entire vault as a structured ZIP
- 🌗 **Dark / Light Mode** — Persistent theme toggle
- 🌐 **English + Hindi UI** — Built-in i18n support
- 📱 **PWA Installable** — Works offline, installable on mobile & desktop
- ♻️ **No Backend** — 100% IndexedDB (localforage), zero data leaves your device

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 19 + Vite (SWC) |
| Storage | localforage (IndexedDB) |
| OCR | Tesseract.js |
| PDF Rendering | pdf.js |
| Icons | Lucide React |
| Export | JSZip + FileSaver |
| Deployment | Vercel |
